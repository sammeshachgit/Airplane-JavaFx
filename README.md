# ✈️ Airline Reservation System

### Efficient Flight Booking & Management System

A desktop-based **Airline Reservation System** developed using **JavaFX** and **MongoDB** to simplify and streamline airline booking operations. The system provides an intuitive interface for managing reservations, passenger details, and flight schedules.

---

## 🚀 Overview

This project is designed to assist airline staff in efficiently handling:

* Flight bookings
* Reservation updates
* Ticket cancellations
* Passenger data management

It ensures a smooth workflow with a **user-friendly GUI** and reliable backend database.

---

## 📌 Features

✨ Create New Flight Reservations
✨ Modify Existing Bookings
✨ Cancel Reservations
✨ Passenger Information Management
✨ Flight Schedule Handling
✨ Interactive JavaFX GUI
✨ Secure and Scalable MongoDB Database

## 🧠 System Workflow

1. User opens the application
2. Selects required operation (Book / Modify / Cancel)
3. Inputs passenger & flight details
4. Data is stored/retrieved from MongoDB
5. System updates records in real-time
